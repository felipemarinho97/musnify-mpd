# musnify-mpd
**Musnify** is a simple script written in python that provides notification support for mpd client using _libnotify_0.7_.

It can also display the album covers loaded from _last.fm_ in the notification.

## Dependencies
To use musnify-mpd you will need to install the following python dependencies:

    $ pip2 install requests mpd

## Use
Just execute the script "musnify-mpd.py" on your autostart or rc.d file and enjoy it. ðŸ˜„
